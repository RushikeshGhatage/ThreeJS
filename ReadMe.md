## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run following commands:

To install dependencies (only the first time)
``` bash
npm install
```
To run the local server at localhost:8080
``` bash
npm start
```
To build for production in the directory
``` bash
npm run build
```
